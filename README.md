# Juego de Parejas (iOS)

Este es un clásico juego de encontrar parejas para iOS, desarrollado con SwiftUI. El objetivo es encontrar todas las parejas de cartas en el menor tiempo posible.

## Características

- **Múltiples Modos de Juego:**
  - **Colores:** Encuentra las parejas de cartas del mismo color.
  - **Letras:** Encuentra las parejas de letras mayúsculas y minúsculas.
  - **Colores y Letras:** Encuentra las parejas de cartas que tienen el mismo color y la misma letra.
  - **Figuras y Colores:** Encuentra las parejas de cartas que tienen la misma figura y el mismo color.
- **Ranking:** Guarda tus mejores tiempos y compite por el top 10 en cada modo de juego.
- **Interfaz Adaptable:** La interfaz se adapta a diferentes tamaños de pantalla, incluyendo iPhone y iPad.
